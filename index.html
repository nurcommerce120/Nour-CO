FAQ

Quels sont les délais de livraison ?

Les commandes sont généralement expédiées sous 3 à 5 jours ouvrés.

Quels moyens de paiement acceptez-vous ?

Vous pouvez payer par PayPal, carte bancaire ou Revolut. Le paiement en espèces est possible uniquement lors d’une remise en main propre dans le 93.

Puis-je suivre ma commande ?

Oui. Une référence de commande vous est fournie afin de suivre l’avancement de votre commande sur la page “Suivi”.

Les frais de livraison sont-ils inclus ?

Non. Les frais de livraison ne sont pas inclus dans le prix affiché et sont ajoutés selon le mode d’envoi choisi.
